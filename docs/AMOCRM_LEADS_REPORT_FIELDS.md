# Отчёт «Потенциальные клиенты» — описание полей

Данные берутся из amoCRM API v4.

## Источник данных

- **Сделки (Leads):** `GET /api/v4/leads` — список сделок с пагинацией
- **Воронки и этапы:** `GET /api/v4/leads/pipelines` — названия воронок и этапов (статусов)
- **Пользователи:** `GET /api/v4/users` — имена ответственных

## Отображаемые поля (колонки отчёта)

| Колонка в отчёте | Поле amoCRM API | Описание |
|------------------|-----------------|----------|
| **ID** | `leads[].id` | Уникальный идентификатор сделки |
| **Название** | `leads[].name` | Название сделки |
| **Бюджет** | `leads[].price` | Бюджет сделки (руб.) |
| **Этап** | `leads[].pipeline_id` + `leads[].status_id` → `pipelines[]._embedded.statuses[]` | Название этапа воронки (по pipeline_id и status_id) |
| **Ответственный** | `leads[].responsible_user_id` → `users[]` | Имя пользователя, ответственного за сделку |
| **Создана** | `leads[].created_at` | Дата и время создания (Unix timestamp) |
| **Обновлена** | `leads[].updated_at` | Дата и время последнего изменения (Unix timestamp) |

## Дополнительные поля в API (можно добавить в отчёт)

| Поле amoCRM | Описание |
|-------------|----------|
| `closed_at` | Дата закрытия сделки |
| `loss_reason_id` | Причина отказа (для проигранных) |
| `source_id` | Источник (с параметром `with=source`) |
| `custom_fields_values` | Произвольные поля |
| `_embedded.tags` | Теги сделки |
| `_embedded.contacts` | Контакты (с `with=contacts`) |
| `_embedded.companies` | Компания (с `with=companies`) |

## Автообновление

Страница отчёта обновляется автоматически каждые **60 секунд** с помощью `<meta http-equiv="refresh" content="60">`.
