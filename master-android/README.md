# СтройКонтроль Мастер — Android-приложение

Мобильное приложение для мастеров системы StroyControl. Отображает веб-интерфейс в WebView.

## Возможности

- Вход под учётной записью мастера
- Просмотр проектов и этапов
- Добавление фотоотчётов (камера и галерея)
- Чат по проекту

## Сборка APK

### Требования

- **Android Studio** Hedgehog (2023.1.1) или новее

### Сборка через Android Studio (рекомендуется)

1. Откройте папку `stroycontrol` или `master-android` в Android Studio
2. Выберите в навигаторе папку `master-android` как корень проекта
3. Дождитесь синхронизации Gradle (автоматически подтянет оболочку Gradle)
4. **Build → Build Bundle(s) / APK(s) → Build APK(s)**
5. APK: `app/build/outputs/apk/debug/app-debug.apk`

### Сборка через командную строку

Если установлен Gradle:

```bash
cd master-android
gradle wrapper
./gradlew assembleDebug
```

APK: `app/build/outputs/apk/debug/app-debug.apk`

### Сборка release APK (для публикации)

1. Создайте keystore и настройте подпись в `app/build.gradle.kts`
2. Выполните: `./gradlew assembleRelease`

## Первый запуск

1. Установите APK на устройство
2. Откройте приложение
3. Нажмите **⋮** (меню) → **Настройки**
4. Введите адрес сервера StroyControl, например:
   - `https://ваш-домен.ru` — если сервер в интернете
   - `http://192.168.1.10:5000` — локальная сеть
5. Сохраните и войдите под логином и паролем мастера

## Важно

- Сервер StroyControl должен быть доступен с телефона (в той же сети или в интернете)
- Для загрузки фото отчётов нужны разрешения «Камера» и «Фото»
- Поддерживается HTTP и HTTPS; для локальной разработки разрешён cleartext (HTTP)
